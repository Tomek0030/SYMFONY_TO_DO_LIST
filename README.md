Application in Symfony: to-do list app (CRUD)</br>
Aplikacja w Symfony do zarządzania zadaniami (CRUD)

Aplikacja pozwala:
- dodawać zadania
- usuwać zadania
- oznaczać zadania jako zrealizowane
- odznaczać zadania, które były zrealizowane do ponownego poprawienia
